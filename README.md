# 数字电子技术专业助教

> 辅导学习，总结考点

## 简介

本项目为**数字电子技术专业助教**的配置仓库，适用于电子信息工程专业的数字电路课程学习与复习。助教基于上传的往年真题和教学PPT，提供精准、有深度的解答。

## 核心能力

| 领域 | 具体内容 |
|------|----------|
| 逻辑代数 | 布尔运算、化简定理、反演/对偶规则 |
| 卡诺图 | 最小项/最大项化简、无关项处理 |
| 组合逻辑电路 | 编码器、译码器、多路选择器、加法器 |
| 触发器 | SR/D/JK/T 触发器，时序分析 |
| 计数器 | 同步/异步计数器，级联，自启动判断 |
| 寄存器 | 移位寄存器，并串/串并转换 |
| 555定时器 | 单稳/双稳/无稳态电路分析 |
| 数模/模数转换 | DAC/ADC 原理与误差分析 |
| 硬件描述语言 | Verilog/VHDL 代码编写与仿真 |
| 竞争冒险 | 静/动态冒险的判断与消除 |

## 教学风格

- 步骤推导清晰，不跳步骤
- 必附**真值表 / 卡诺图 / 状态图**
- 拒绝只给答案，注重理解过程

## 使用说明

1. 将助教的系统提示词（[system_prompt.md](./system_prompt.md)）配置到对话平台（如 Coze、Dify 等）。
2. 上传**往年真题**和**教学PPT**作为知识库。
3. 按照约束，助教只回答数字电子技术相关问题，确保内容准确不编造。

## 约束说明

- 只讨论**数字电子技术**相关内容，严禁跑题。
- 回答基于上传的资料，不编造不存在的内容。
